✨ 核心功能 (Features)

基于原版逻辑进行了深度重构与汉化，专门为中文桌游玩家打造：

🎲 多种抽取模式：

纯随机 (Pure Random)：最快开局，系统自动分配。

三选二 (Pick 2 of 3)：交互式轮抽，每位玩家从 3 张手牌中弃掉 1 张。

蛇形轮抽 (Snake Draft)：公平的 1-2-3-4-4-3-2-1 顺序，支持自定义每次展示数量。

👑 昆斯伯里模式 (Queensberry)：独家新增！所有候选阵营摊开在公共池中，玩家轮流抢夺。

🛡️ 战术 BP 系统：

支持在轮抽中加入 Ban (禁用) / Pick (选用) 环节。

支持 穿插式 BP (Interleaved)：Ban -> Pick -> Ban -> Pick，还原电竞级博弈体验。

⚙️ 灵活配置：

自定义公共池大小（$2n+x$）。

支持将未选阵营“移出游戏”或“洗回牌堆”。

随机决定起始玩家（定庄）。

🛠️ 本地运行 (Local Development)

如果你想在没有网络的环境下使用，或者想修改代码：

本项目为单文件 HTML 架构，无需安装任何环境。

方式一（推荐）： * 在本仓库页面点击绿色的 <> Code 按钮，选择 Download ZIP。

解压后，直接双击文件夹中的 index.html 即可运行。

方式二（Git）：

git clone [https://github.com/deathcats4/-Smash-Up-Randomizer-CN-.git](https://github.com/deathcats4/-Smash-Up-Randomizer-CN-.git)


然后进入目录双击 index.html。
